데이터베이스 및 데이터소스 리소스를 관리합니다.

## 엔드포인트

- **GET** `/rest/resource/databases` — 데이터베이스 목록 조회
- **GET** `/rest/resource/databases/{resourceName}` — 데이터베이스 상세 조회
- **POST** `/rest/resource/databases` — 데이터베이스 생성
- **DELETE** `/rest/resource/databases/{resourceName}` — 데이터베이스 삭제
- **GET** `/rest/resource/datasources` — 데이터소스 목록 조회
- **POST** `/rest/resource/datasources` — 데이터소스 생성
- **GET** `/rest/resource/datasources/{resourceName}` — 데이터소스 상세 조회
- **PUT** `/rest/resource/datasource/{resourceName}` — 데이터소스 수정
- **DELETE** `/rest/resource/datasources/{resourceName}` — 데이터소스 삭제
