시스템의 생성, 조회, 수정, 삭제를 관리합니다.

## 엔드포인트

- **GET** `/rest/systems` — 모든 시스템 조회
- **GET** `/rest/systems/{systemName}` — 시스템 상세 조회
- **POST** `/rest/systems` — 시스템 생성
- **PUT** `/rest/systems/{systemName}` — 시스템 수정
- **DELETE** `/rest/systems/{systemName}` — 시스템 삭제
